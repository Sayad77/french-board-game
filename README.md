# French-board-game (Prototype)

Prototype d'un jeu de plateau éducatif pour apprendre le français (vocabulaire + grammaire).
Ce dépôt contient :
- backend Express (server/)
- prototype console (console-client/)
- frontend minimal (web-client/)
- fichiers de données JSON (server/data/)

---

## Installation & lancement local (recommandé pour le prototype)

1. Clone le repo ou crée les dossiers localement.
2. Installer le backend :
   ```bash
   cd server
   npm install
   node index.js
